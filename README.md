## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/juansilvadesign/juansilva.is-a.dev.git
cd juansilva.is-a.dev
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for static export
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Built With

- **Next.js 14** - React framework for production
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **React Context API** - State management
- **Radix UI** - Unstyled, accessible components
- **Responsive Design** - Mobile-first approach

## Architecture

- **App Router** - Next.js 14 app directory structure
- **Server/Client Components** - Optimized rendering strategy
- **Static Export** - Pre-rendered HTML output to `dist/` folder
- **Context Provider** - Landing quiz state management across components

## Deployment

Builds to static HTML and deploys to custom domain at `juansilva.is-a.dev`

```bash
npm run build
```

Output is generated in the `dist/` directory, ready for static hosting.
